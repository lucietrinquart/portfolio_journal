OX FEST FESTIVAL: UNE MAQUETTE IMMERSIVE

Un projet créatif de conception web pour un festival imaginaire

Le Projet en Bref

Nous avions pour mission de concevoir une maquette complète pour un festival imaginaire nommé "OX Fest Festival". Ce projet devait être réalisé en anglais, avec une liberté créative totale quant aux technologies à utiliser.

J'ai choisi de relever ce défi en utilisant HTML, CSS et JavaScript pour créer une expérience web immersive et interactive. Cette approche m'a permis de donner vie à ma vision tout en démontrant mes compétences techniques en développement front-end.

La première étape fut une recherche approfondie d'inspiration, suivie par la conception d'une maquette initiale, que j'ai ensuite affinée jusqu'à obtenir le résultat final que vous pouvez découvrir ci-dessous.

Points clés du projet
  • Conception d'une maquette complète pour un festival fictif
  • Réalisation en HTML, CSS et JavaScript
  • Contenu entièrement en anglais
  • Design responsive pour tous les appareils
  • Interface utilisateur intuitive et immersive
N'oubliez pas de mentionner l'aspect responsive, c'est crucial pour l'évaluation!

Mes Sources d'Inspiration

Pour créer une expérience véritablement unique, je me suis inspirée de différents univers visuels et tendances de design contemporain.
