LE POKÉDEX: UN PROJET COMPLET DE DÉVELOPPEMENT WEB

Une application moderne combinant Laravel, Angular et Tailwind

Présentation du projet

Le projet Pokédex représente une étape importante dans mon parcours de développement web. Il s'agissait de créer une application complète en utilisant des technologies modernes: Laravel pour le backend, Angular pour le frontend et Tailwind CSS pour le style.

Chaque étudiant a reçu une maquette unique sur le thème Pokémon, qu'il devait reproduire fidèlement. Notre application devait consommer une API Pokémon pour afficher les données des créatures, et implémenter un système d'authentification via GitHub.

Ce projet complet nous a permis d'explorer l'intégration de différentes technologies et de comprendre comment elles fonctionnent ensemble pour créer une application web moderne et fonctionnelle.

Difficultés rencontrées

Complexité de l'API Pokémon
Maquette incomplète et imprécise
❮ ❯

Méconnaissance de l'univers Pokémon: N'étant pas familière avec cet univers, j'ai eu du mal à comprendre certains aspects de l'API et à saisir la logique derrière les relations entre les différentes entités.

Maquette incomplète: La maquette qui m'a été fournie comportait des éléments manquants ou imprécis, ce qui a compliqué son implémentation fidèle.

Authentification GitHub: L'implémentation de l'authentification via GitHub a été particulièrement problématique, avec de nombreuses erreurs en boucle qui ont nécessité une recherche approfondie pour être résolues.

Je suis fière de voir ma progression à travers ces projets de plus en plus complexes!
